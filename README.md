# How To Run

## Installation

Install my-project with npm

1. Make Environment for Flask App (backend)
```bash
  py -3 -m venv .venv
  .venv\Scripts\activate
```

2. Install required library
```bash
  pip install Flask
  pip install flask_cors
```

3. Run the project
```bash
  python app.py
```
4. Open the templates/index.html file on browser
